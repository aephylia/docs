All file operations are **sandboxed** to `Documents/aephylia/workspace/`. You can't escape this directory — paths with `..` or `:` are blocked automatically.

<Warning>
  All paths are relative to the workspace root. Trying to use `..` to go up or absolute paths with `:` will throw an error.
</Warning>

---

## Reading

<ResponseField name="readfile(path)" type="string">
  Reads the entire contents of a file as a string. Throws an error if the file doesn't exist.

  <ParamField path="path" type="string" required>Relative path inside your workspace</ParamField>

  ```lua
  local data = readfile("config.json")
  local config = JSONDecode(data)
  print(config.name)
  ```
</ResponseField>

---

## Writing

<ResponseField name="writefile(path, content)" type="void">
  Writes content to a file. Creates it if it doesn't exist, **overwrites** it if it does.

  <ParamField path="path" type="string" required>Relative path</ParamField>
  <ParamField path="content" type="string" required>What to write</ParamField>

  ```lua
  local config = { fov = 150, smooth = 5 }
  writefile("config.json", JSONEncode(config))
  ```
</ResponseField>

<ResponseField name="appendfile(path, content)" type="void">
  Adds content to the end of an existing file — doesn't overwrite what's already there.

  <ParamField path="path" type="string" required>Relative path</ParamField>
  <ParamField path="content" type="string" required>Content to append</ParamField>

  ```lua
  appendfile("log.txt", "Script started at " .. os.date() .. "\n")
  ```
</ResponseField>

## Checking

<ResponseField name="isfile(path)" type="boolean">
  Returns `true` if the path points to a file that exists.

  <ParamField path="path" type="string" required>Relative path</ParamField>
</ResponseField>

<ResponseField name="isfolder(path)" type="boolean">
  Returns `true` if the path points to a directory that exists.

  <ParamField path="path" type="string" required>Relative path</ParamField>
</ResponseField>

```lua
if isfile("config.json") then
    print("Config exists!")
else
    print("No config found, creating defaults...")
end
```

## Directories

<ResponseField name="makefolder(path)" type="void">
  Creates a directory (and any parent directories that don't exist yet).

  <ParamField path="path" type="string" required>Relative path</ParamField>

  ```lua
  makefolder("saves/backups")
  ```
</ResponseField>

<ResponseField name="delfolder(path)" type="void">
  Recursively deletes a directory and everything inside it. Be careful with this one!

  <ParamField path="path" type="string" required>Relative path</ParamField>
</ResponseField>

## Listing

<ResponseField name="listfiles(path)" type="table">
  Returns an array of all files and folders inside a directory. Paths come back as relative paths with `/` separators. Throws an error if the path isn't a directory.

  <ParamField path="path" type="string" required>Relative directory path</ParamField>

  ```lua
  local files = listfiles("saves")
  for _, file in ipairs(files) do
      print(file)
  end
  -- saves/save1.json
  -- saves/save2.json
  ```
</ResponseField>

## Full Example — Config System

```lua
local CONFIG_FILE = "my_config.json"

local defaults = {
    aimbot = { enabled = false, fov = 150, smooth = 5 },
    esp = { enabled = true, boxes = true, names = true },
}

-- Load existing or create fresh
local config
if isfile(CONFIG_FILE) then
    config = JSONDecode(readfile(CONFIG_FILE))
    notify("Config loaded", "Config", 2, 1)
else
    config = defaults
    writefile(CONFIG_FILE, JSONEncode(defaults))
    notify("Default config created", "Config", 2, 0)
end

-- Apply it
if config.aimbot.enabled then
    Aimbot.Enable()
    Aimbot.SetFOV(config.aimbot.fov)
    Aimbot.SetSmoothing(config.aimbot.smooth)
end

if config.esp.enabled then
    ESP.Enable()
    ESP.SetBoxes(config.esp.boxes)
    ESP.SetNames(config.esp.names)
end
```
