---

## Silent Aim

Silent aim works differently from the regular aimbot. Instead of moving your mouse, it modifies the data sent to the server so your shots land on the target without visibly aiming at them. Your crosshair can be pointing anywhere — shots still hit.

<ResponseField name="Silent.Enable()" type="void">
  Turn on silent aim.
</ResponseField>

<ResponseField name="Silent.Disable()" type="void">
  Turn off silent aim.
</ResponseField>

<ResponseField name="Silent.Toggle()" type="boolean">
  Flip silent aim on or off. Returns the new state.
</ResponseField>

<ResponseField name="Silent.IsEnabled()" type="boolean">
  Check if silent aim is on.
</ResponseField>

<ResponseField name="Silent.SetFOV(value)" type="void">
  Set how far from your crosshair silent aim will look for targets.
  <ParamField path="value" type="number" required>
    FOV radius in pixels (e.g. `200`)
  </ParamField>
</ResponseField>

<ResponseField name="Silent.GetFOV()" type="number">
  Returns the current silent aim FOV radius.
</ResponseField>

## Desync

Desync makes your character's server-side position different from what other players see. In other words, where enemies shoot at isn't actually where your character is on the server — making you much harder to hit.

<ResponseField name="Desync.Enable()" type="void">
  Turn on desync.
</ResponseField>

<ResponseField name="Desync.Disable()" type="void">
  Turn off desync.
</ResponseField>

<ResponseField name="Desync.Toggle()" type="boolean">
  Flip desync on or off. Returns the new state.
</ResponseField>

<ResponseField name="Desync.IsEnabled()" type="boolean">
  Check if desync is currently on.
</ResponseField>

## Tickrate

The tickrate controls how often the game's physics simulation updates. Roblox normally runs at 60 ticks per second — cranking this up makes your character move and act faster than everyone else.

<ResponseField name="Tickrate.Enable()" type="void">
  Turn on tickrate override.
</ResponseField>

<ResponseField name="Tickrate.Disable()" type="void">
  Turn off tickrate override (goes back to default).
</ResponseField>

<ResponseField name="Tickrate.Set(value)" type="void">
  Set the tickrate value **and** enable it in one call.
  <ParamField path="value" type="number" required>
    Ticks per second (e.g. `240`). Default Roblox is `60`.
  </ParamField>
</ResponseField>

<ResponseField name="Tickrate.Get()" type="number">
  Returns the current tickrate value.
</ResponseField>

<ResponseField name="Tickrate.IsEnabled()" type="boolean">
  Check if tickrate override is active.
</ResponseField>

## Full Example — Combat Setup

```lua
-- Turn everything on
Silent.Enable()
Silent.SetFOV(200)
Desync.Enable()
Tickrate.Set(240)

notify("Combat mode ON", "Script", 2, 1)

-- Toggle everything with one key
while true do
    if Input.IsKeyPressed(0x75) then -- F6
        local silent = Silent.Toggle()
        if silent then
            Desync.Enable()
            Tickrate.Set(240)
            notify("Combat ON", "Combat", 2, 1)
        else
            Desync.Disable()
            Tickrate.Disable()
            notify("Combat OFF", "Combat", 2, 2)
        end
    end
    wait(0.01)
end
```
