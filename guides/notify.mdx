aephylia has a built-in notification system that shows animated toast cards on your overlay. They slide in, show a progress countdown, and disappear on their own. Here's how to use them from scripts.

## Basic Usage

```lua
notify("Hello from my script!", "My Script", 3, 0)
```

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `message` | string | — | The notification body text |
| `title` | string | `"aephylia"` | Bold header text |
| `duration` | number | `5.0` | How long it stays visible (seconds) |
| `type` | number | `0` | Accent color style (see below) |

---

## Types

Each type gets a different colored accent bar so you can tell at a glance what kind of notification it is:

| Type | Code | Color | When to use |
|------|------|-------|-------------|
| Info | `0` | Accent color | General information |
| Success | `1` | Green | Something worked — toggles on, configs loaded |
| Error | `2` | Red | Something failed — load errors, missing files |
| Warning | `3` | Amber | Heads up — low health, suspicious player nearby |

```lua
notify("Script started", "Info", 3, 0)      -- accent
notify("Aimbot enabled", "Combat", 2, 1)     -- green
notify("Config not found", "Error", 5, 2)    -- red
notify("Low health!", "Warning", 4, 3)       -- amber
```

---

## Stacking

Multiple notifications stack vertically. New ones appear and push existing ones down with a smooth animation. They all have independent timers so they fade out individually.

```lua
notify("First", "Stack", 5, 0)
wait(0.3)
notify("Second", "Stack", 5, 1)
wait(0.3)
notify("Third", "Stack", 5, 2)
```

---

## Auto-Notifications

When you toggle built-in features using their API (like `Aimbot.Toggle()`), aephylia automatically shows a notification for you. You don't need to manually call `notify()` for these:

```lua
-- These all show notifications automatically
Aimbot.Toggle()
ESP.Toggle()
Silent.Toggle()
Flight.Toggle()
Noclip.Toggle()
Desync.Toggle()
```

## Practical Patterns

### Script Startup

```lua
notify("BHop loaded", "Script", 3, 1)
notify("Press F4 to toggle", "Script", 3, 0)
```

### Error Handling

```lua
local ok, err = pcall(function()
    local data = JSONDecode(readfile("config.json"))
end)

if not ok then
    notify("Config error: " .. tostring(err), "Error", 5, 2)
end
```

### State Feedback

```lua
while true do
    if Input.IsKeyPressed(0x74) then -- F5
        local on = ESP.Toggle()
        -- Module auto-notifies, but you can add extra info:
        if on then
            notify("Distance + Names enabled", "ESP", 2, 0)
        end
    end
    wait(0.01)
end
```
