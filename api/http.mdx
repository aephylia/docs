Simple HTTP functions for fetching data from the web or posting to APIs. Both support HTTPS and use WinINet under the hood.

---

## Functions

<ResponseField name="HttpGet(url)" type="string">
  Performs an HTTP GET request and returns the response body as a string. Throws a Lua error if the request fails.

  <ParamField path="url" type="string" required>The URL to fetch</ParamField>

  ```lua
  local body = HttpGet("https://httpbin.org/get")
  print(body)
  ```
</ResponseField>

<ResponseField name="HttpPost(url, body, contentType?)" type="string | nil">
  Performs an HTTP POST request. Returns the response body, or `nil` if something goes wrong.

  <ParamField path="url" type="string" required>The URL to post to</ParamField>
  <ParamField path="body" type="string" required>Request body content</ParamField>
  <ParamField path="contentType" type="string" default="application/json">The Content-Type header</ParamField>

  ```lua
  local payload = JSONEncode({ event = "started", time = os.time() })
  local response = HttpPost("https://example.com/api/log", payload)
  if response then
      print("Server said:", response)
  end
  ```
</ResponseField>

## Examples

### Fetch JSON Data

```lua
local raw = HttpGet("https://api.example.com/data.json")
local data = JSONDecode(raw)
print("Server says:", data.message)
```

### Discord Webhook

```lua
local function webhook(url, content)
    local payload = JSONEncode({ content = content })
    HttpPost(url, payload, "application/json")
end

webhook("https://discord.com/api/webhooks/...", "Script started!")
```

### Load Remote Script

```lua
local code = HttpGet("https://raw.githubusercontent.com/user/repo/main/script.lua")
local fn, err = loadstring(code)
if fn then
    fn()
else
    notify("Failed: " .. err, "Loader", 5, 2)
end
```
