The movement APIs give you control over flight and noclip — fly around the map or walk through walls.

---

## Flight

Enables free flight, letting you move in any direction and ignore gravity.

<ResponseField name="Flight.Enable()" type="void">
  Turn on flight.
</ResponseField>

<ResponseField name="Flight.Disable()" type="void">
  Turn off flight.
</ResponseField>

<ResponseField name="Flight.Toggle()" type="boolean">
  Flip flight on or off. Returns the new state.
</ResponseField>

<ResponseField name="Flight.IsEnabled()" type="boolean">
  Check if flight is currently on.
</ResponseField>

<ResponseField name="Flight.SetSpeed(speed)" type="void">
  Set how fast you fly.
  <ParamField path="speed" type="number" required>
    Flight speed value — higher = faster.
  </ParamField>
</ResponseField>

<ResponseField name="Flight.GetSpeed()" type="number">
  Returns the current flight speed.
</ResponseField>

```lua
-- Toggle flight with F7
while true do
    if Input.IsKeyPressed(0x76) then -- F7
        local on = Flight.Toggle()
        if on then
            Flight.SetSpeed(100)
            notify("Flight ON", "Movement", 2, 1)
        else
            notify("Flight OFF", "Movement", 2, 2)
        end
    end
    wait(0.01)
end
```

---

## Noclip

Walk straight through walls, floors, and terrain by disabling collision.

<ResponseField name="Noclip.Enable()" type="void">
  Turn on noclip.
</ResponseField>

<ResponseField name="Noclip.Disable()" type="void">
  Turn off noclip.
</ResponseField>

<ResponseField name="Noclip.Toggle()" type="boolean">
  Flip noclip on or off. Returns the new state.
</ResponseField>

<ResponseField name="Noclip.IsEnabled()" type="boolean">
  Check if noclip is currently on.
</ResponseField>

```lua
-- Toggle noclip with F8
while true do
    if Input.IsKeyPressed(0x77) then -- F8
        local on = Noclip.Toggle()
        notify(on and "Noclip ON" or "Noclip OFF", "Movement", 2, on and 1 or 2)
    end
    wait(0.01)
end
```

---

## Combining Both

Flight + noclip together lets you fly through walls — useful for exploring maps or getting to hard-to-reach spots.

```lua
-- F7 = flight, F8 = noclip, F9 = both
while true do
    if Input.IsKeyPressed(0x76) then -- F7
        Flight.Toggle()
    end
    if Input.IsKeyPressed(0x77) then -- F8
        Noclip.Toggle()
    end
    if Input.IsKeyPressed(0x78) then -- F9
        local fly = Flight.Toggle()
        if fly then Noclip.Enable() else Noclip.Disable() end
        notify(fly and "Ghost mode ON" or "Ghost mode OFF", "Movement", 2, fly and 1 or 2)
    end
    wait(0.01)
end
```
