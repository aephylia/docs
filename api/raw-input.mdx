These functions **simulate** physical input events. For **reading** key states, see the [Input](/api/input) module.

<Warning>
  Simulated input sends real OS-level events. Use responsibly — rapid input may trigger anti-cheat detection in some games.
</Warning>

---

## Mouse Movement

<ResponseField name="mousemoverel(x, y)" type="void">
  Moves the mouse cursor by a **relative** offset in pixels.

  <ParamField path="x" type="number" required>Horizontal offset (positive = right)</ParamField>
  <ParamField path="y" type="number" required>Vertical offset (positive = down)</ParamField>

  ```lua
  mousemoverel(10, -5) -- move right 10px, up 5px
  ```
</ResponseField>

<ResponseField name="mousemoveabs(x, y)" type="void">
  Moves the mouse cursor to an **absolute** screen position.

  <ParamField path="x" type="number" required>Screen X coordinate</ParamField>
  <ParamField path="y" type="number" required>Screen Y coordinate</ParamField>

  ```lua
  local size = Game.GetScreenSize()
  mousemoveabs(size.Width / 2, size.Height / 2) -- move to center
  ```
</ResponseField>

## Mouse Clicks

<ResponseField name="mouseclick(button)" type="void">
  Simulates a mouse button press and release.

  <ParamField path="button" type="number" default="0">
    - `0` — Left click
    - `1` — Right click
    - `2` — Middle click
  </ParamField>

  ```lua
  mouseclick()    -- left click
  mouseclick(1)   -- right click
  mouseclick(2)   -- middle click
  ```
</ResponseField>

---

## Mouse Scroll

<ResponseField name="mousescroll(amount)" type="void">
  Simulates a mouse scroll event.

  <ParamField path="amount" type="number" required>
    Scroll amount. Positive = scroll up, negative = scroll down.
  </ParamField>

  ```lua
  mousescroll(3)   -- scroll up
  mousescroll(-3)  -- scroll down
  ```
</ResponseField>

## Keyboard

<ResponseField name="keypress(keycode)" type="void">
  Simulates a key press and release using [Virtual Key Codes](https://learn.microsoft.com/en-us/windows/win32/inputdev/virtual-key-codes).

  <ParamField path="keycode" type="number" required>Virtual key code</ParamField>

  ```lua
  keypress(0x57)  -- press 'W'
  keypress(0x20)  -- press Space
  keypress(0x0D)  -- press Enter
  ```
</ResponseField>

---

## Common Key Codes

| Key | Code | Key | Code |
|-----|------|-----|------|
| W | `0x57` | A | `0x41` |
| S | `0x53` | D | `0x44` |
| Space | `0x20` | Shift | `0x10` |
| Enter | `0x0D` | Escape | `0x1B` |
| Tab | `0x09` | Ctrl | `0x11` |
| 0-9 | `0x30`–`0x39` | F1-F12 | `0x70`–`0x7B` |

## Example — Auto-Jump

```lua
-- Simple bunny hop: press space when grounded
while true do
    if isrbxactive() and Input.IsKeyDown(0x20) then
        keypress(0x20)
    end
    wait(0.005)
end
```
