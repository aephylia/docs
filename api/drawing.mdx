description: "Persistent object-oriented drawing API"

---

The Drawing API lets you create **persistent** drawing objects that stay on screen until you explicitly remove them. No need to redraw every frame — just create an object, set its properties, and it'll keep rendering on its own.

If you've used other Roblox executor drawing APIs before, this should feel familiar — it follows the same `Drawing.new()` pattern.

<Note>
  For quick frame-by-frame drawing (stuff that gets wiped every frame), use the [Draw (Immediate)](/api/draw) API instead.
</Note>

---

## Creating Objects

<ResponseField name="Drawing.new(type)" type="DrawingObject">
  Creates a new persistent drawing object. Objects start invisible — set `Visible = true` to show them.

  <ParamField path="type" type="string" required>
    One of: `"Line"`, `"Circle"`, `"Square"`, `"Text"`, `"Triangle"`, `"Quad"`
  </ParamField>

  ```lua
  local line = Drawing.new("Line")
  line.From = Vector2.new(100, 100)
  line.To = Vector2.new(400, 400)
  line.Color = Color3.fromRGB(255, 0, 0)
  line.Thickness = 2
  line.Visible = true
  ```
</ResponseField>

<ResponseField name="Drawing.clear()" type="void">
  Removes **every** persistent drawing object at once. Good for cleanup.
</ResponseField>

---

## Shared Properties

These work on every type of drawing object:

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `Visible` | boolean | `false` | Whether the object actually gets drawn |
| `Color` | Color3 | `{0, 0, 0}` | Stroke or fill color |
| `Transparency` | number | `1.0` | Opacity — `1` is fully visible, `0` is invisible |
| `ZIndex` | number | `0` | Draw order (higher = rendered on top) |
| `Thickness` | number | `1.0` | Line thickness for outlines |
| `Filled` | boolean | `false` | Whether shapes are filled or just outlines |

---

## Methods

Every drawing object has these:

<ResponseField name="object:Remove()" type="void">
  Destroys the drawing object and removes it from the screen.
</ResponseField>

<ResponseField name="object:Destroy()" type="void">
  Same as `Remove()` — just an alias.
</ResponseField>

## Line

| Property | Type | Description |
|----------|------|-------------|
| `From` | Vector2 | Start point |
| `To` | Vector2 | End point |

```lua
local line = Drawing.new("Line")
line.From = Vector2.new(0, 0)
line.To = Vector2.new(500, 500)
line.Color = Color3.fromRGB(255, 255, 0)
line.Thickness = 2
line.Visible = true
```

---

## Circle

| Property | Type | Description |
|----------|------|-------------|
| `Position` | Vector2 | Center point |
| `Radius` | number | Circle radius in pixels |
| `NumSides` | number | Segment count (more = smoother, `0` = auto) |

```lua
local circle = Drawing.new("Circle")
circle.Position = Vector2.new(400, 300)
circle.Radius = 50
circle.Color = Color3.fromRGB(0, 255, 0)
circle.NumSides = 64
circle.Filled = false
circle.Visible = true
```

---

## Square

| Property | Type | Description |
|----------|------|-------------|
| `Position` | Vector2 | Top-left corner |
| `Size` | Vector2 | Width and height |

```lua
local box = Drawing.new("Square")
box.Position = Vector2.new(100, 100)
box.Size = Vector2.new(200, 150)
box.Color = Color3.fromRGB(0, 120, 255)
box.Filled = true
box.Visible = true
```

## Text

| Property | Type | Description |
|----------|------|-------------|
| `Position` | Vector2 | Text position |
| `Text` | string | The text content |
| `Size` | number | Font size |
| `Center` | boolean | Center-align horizontally around the position |
| `Outline` | boolean | Draw a dark outline behind the text |
| `OutlineColor` | Color3 | Color of the outline |
| `Font` | string | Font name (placeholder — uses default font) |

```lua
local text = Drawing.new("Text")
text.Position = Vector2.new(200, 50)
text.Text = "Hello aephylia!"
text.Size = 18
text.Color = Color3.fromRGB(255, 255, 255)
text.Center = true
text.Outline = true
text.OutlineColor = Color3.fromRGB(0, 0, 0)
text.Visible = true
```

## Triangle

| Property | Type | Description |
|----------|------|-------------|
| `PointA` | Vector2 | First vertex |
| `PointB` | Vector2 | Second vertex |
| `PointC` | Vector2 | Third vertex |

```lua
local tri = Drawing.new("Triangle")
tri.PointA = Vector2.new(300, 100)
tri.PointB = Vector2.new(200, 300)
tri.PointC = Vector2.new(400, 300)
tri.Color = Color3.fromRGB(255, 128, 0)
tri.Filled = true
tri.Visible = true
```

## Quad

| Property | Type | Description |
|----------|------|-------------|
| `PointA` | Vector2 | First vertex |
| `PointB` | Vector2 | Second vertex |
| `PointC` | Vector2 | Third vertex |
| `PointD` | Vector2 | Fourth vertex |

```lua
local quad = Drawing.new("Quad")
quad.PointA = Vector2.new(100, 100)

quad.PointC = Vector2.new(350, 300)
quad.PointD = Vector2.new(50, 300)
quad.Color = Color3.fromRGB(128, 0, 255)
quad.Filled = true
quad.Visible = true
```

## Full Example — Animated Crosshair

A crosshair that stays on screen without any loop — just create the objects and you're done:

```lua
local size = Game.GetScreenSize()
local cx, cy = size.Width / 2, size.Height / 2

local lines = {}
local offsets = {
    { -10, 0, -3, 0 }, -- left
    { 3, 0, 10, 0 },   -- right
    { 0, -10, 0, -3 }, -- top
    { 0, 3, 0, 10 },   -- bottom
}

for _, o in ipairs(offsets) do
    local l = Drawing.new("Line")
    l.From = Vector2.new(cx + o[1], cy + o[2])
    l.To = Vector2.new(cx + o[3], cy + o[4])
    l.Color = Color3.fromRGB(0, 255, 0)
    l.Thickness = 2
    l.Visible = true
    table.insert(lines, l)
end

-- Center dot
local dot = Drawing.new("Circle")
dot.Position = Vector2.new(cx, cy)
dot.Radius = 2
dot.Color = Color3.fromRGB(0, 255, 0)
dot.Filled = true
dot.Visible = true
```

<Tip>
  OOP drawings persist across frames — you don't need to recreate them in a loop. Just call `:Remove()` when you're done with an object to free resources.
</Tip>
