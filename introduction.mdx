## What is the aephylia Lua API?

aephylia embeds a high-performance **LuaJIT** runtime that lets you write custom scripts to automate features, create overlays, react to game events, and build entirely new functionality on top of the engine. LuaJIT compiles Lua to native machine code, giving scripts near-C performance.

Scripts run in **background threads** with their own isolated state. Each script can be started, stopped, and edited independently from the built-in script editor.

## Quick example

```lua
-- Toggle ESP with F2, show a notification
while true do
    if Input.IsKeyPressed(0x71) then
        local state = ESP.Toggle()
        notify(state and "ON" or "OFF", "ESP", 1.5, state and 1 or 2)
    end
    wait(0.01)
end
```
